### How to run

Install Redis on Mac through [Homebrew] or Windows through [Chocolatey]

```sh
$ brew install redis

$ choco install redis-64
```

Then run the following command.

```sh
$ npm i
$ redis-server
$ node index.js
$ node server.js
```
